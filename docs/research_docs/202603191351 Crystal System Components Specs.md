## Connection Breakdown for Building

If you are building an interface or a custom resistivity tester, the "64/7s" follows the standard **Kelvin (4-Wire) Sensing** layout.

|**Connection**|**Function**|**Description**|
|---|---|---|
|**Outer Pin 1**|**Current Source (+)**|Injects the known current into the sample.|
|**Inner Pin 2**|**Voltage Sense (+)**|Measures the high-side potential.|
|**Inner Pin 3**|**Voltage Sense (-)**|Measures the low-side potential.|
|**Outer Pin 4**|**Current Sink (-)**|Completes the current loop to ground/return.|
